description=[url=https://steamcommunity.com/sharedfiles/filedetails/?id=3180893708]Mod version for build 41 of Project Zomboid[/url]
description=
description=This mod changes the logic of the automatic sorting of mods added in build 42 of ProjectZomboid.
description=Now the automatic sorting supports sorting rules, as well as highlights all found violations of the mod loading order.
description=
description=[h3]How to use:[/h3]
description=All you need to do is enable this mod and apply the changes. Then activate all the mods you want, go to the mod load order and click the "Sort" button at the bottom of the screen. Apply the changes.
description=Done!
description=If necessary, you can manually change the mod load order or set your own rules using the sorting rules editor.
description=
description=[h3]Warning:[/h3]
description=Any change in the order of loading mods in the middle of the game can lead to loss of the current save. Do so at your own risk.
description=
description=[h3]Features:[/h3]
description=[list]
description=[*]Automatic sorting:
description= The list of Active mods is pre-sorted alphabetically and also grouped in the following order:
description=[olist]
description=[*]Framework/API/Tweak mods,
description=[*]Resources/Textures,
description=[*]Maps/Locations,
description=[*]Vechicles,
description=[*]Code-only mods,
description=[*]Clothing,
description=[*]Interface,
description=[*]Other mods,
description=[*]Localization
description=[/olist]
description= Subsequent application of the [b]topological sorting algorithm[/b] can detect and correct incorrect order of mods according to the requirements of each mod.
description=[*]Support for [b]Sorting Rules[/b]. Sorting Rules Editor
description=[*]Highlighting mods with detected violations
description=[*][b]Search[/b] by mod name or mod ID
description=[*]Support for [b]multiple file selection[/b] to manually edit the order of loading mods.
description=[*]When running the game in debug mode, additional information about the mod on which the mouse cursor is hovered will be displayed.
description=[/list]
description=
description=[h3]Disclamer[/h3]
description=This mod is not a cure for all conflicts in the order of loading mods. Some conflicts can only be resolved manually.
description=

This version of the mod is only for version b42. For version 41 of the game use the mod of the corresponding version https://steamcommunity.com/sharedfiles/filedetails/?id=3180893708
